Пояснения к task 2
Изменения переменной PATH, выполненные в терминале, действуют только в текущей сессии оболочки, а при закрытии терминала они теряются, потому что не было сохранения в конфигурационных файлах. Чтобы изменения стали постоянными, нужно добавить строку (напишу ниже) с обновлением PATH в файл .bashrc﻿:
Строка:
export PATH="$PATH:Documents/VS Code"
После редактирования .bashrc нужно применить изменения. Для этого просто закрываю и снова открываю терминал

Пояснения к task 7
Написала команду alias, сначала она будет временной и после завершения активного сеанса больше не будет работать, если её заново не прописать. Чтобы сделать ее постоянной нужно добавить в файл действующей интерактивной оболочки (.bashrc). Теперь и после перезагрузки терминала можно вызывать этот alias и он будет срабатывать.
